# WhatNext Vision Motors - Salesforce Capstone Project

## Project Overview
A comprehensive Salesforce CRM implementation for streamlining vehicle sales operations, including inventory management, automated dealer assignment, and customer engagement.

## Repository Contents
- `SalesforceCapstone_WhatNextVisionMotors.pdf` - Complete project documentation
- `/screenshots` - All application screenshots and test cases
- `README.md` - This file

## Key Features
- Custom Objects for Vehicles, Orders, Customers, Dealers
- Automated Flows for dealer assignment and test drive reminders
- Apex Triggers for stock validation and inventory management
- Batch Apex for bulk order processing
- Lightning App with custom UI
